## 关于我

来自中国大陆的 Minecraft 生态开发者，专注于将经典模组以 Spigot/Paper 插件的形式重新实现，同时打造桌面端资源包编辑器等开发工具。也是一名 Bilibili UP主，分享游戏内容与技术教程。

- **核心方向** — Minecraft 插件开发 / 桌面应用 (Tauri) / 开源社区
- **技术栈** — `Java` `TypeScript` `JavaScript` `Python` `Rust (Tauri)`
- **文档站** — [docs.little100.cn](https://docs.little100.cn/)
